# Hackathon Context

This folder stores copied public hackathon reference materials that should guide PadalaSplit planning and implementation.

Use this folder for:

- Hackathon setup guides
- README or submission format requirements
- Checkpoint instructions
- Judging criteria
- Public deployment and demo requirements

Do not place secrets here. Never commit seed phrases, private keys, API tokens, funded wallet credentials, or private team notes.

Future agents should read the files in this folder directly instead of refetching the same GitHub references. Add new hackathon formats or guides here when they should become part of the public repository context.

## Included References

- `readme-format.md` - copied README structure required for hackathon project submissions.
- `stellar-soroban-mainnet-guide.md` - local summary and source link for the Stellar Hackathon PH 2026 deployment guide.
